<b> <u> Weitere Sicherungserweiterungen </u> </b> - Zusätzliche Unterstützung für Teilsicherungen. Sie können nun eine bestimmte Kategorie sichern, indem Sie auf die Kategorie klicken und "Kategorie sichern" auswählen.
<br>
<b> <u> Weitere Wiederherstellungserweiterungen </u> </b> - Möglichkeit zur Wiederherstellung von Backups ohne überschreiben vorhandener Daten. Sie können nun ein Backup in eine bestehende Kategorie wiederherstellen, indem Sie auf die Kategorie klicken und "Backup wiederherstellen" auswählen.
