<b> <u> Backup-Erweiterungen </u> </b> - Die Möglichkeit, mehrere Backup-Dateien zu haben, wurden hinzugefügt. Wenn Sie eine Sicherung durchführen, können Sie nun den Dateinamen angeben. Sie können so viele Backup-Dateien wie Ihr Gerät ermöglicht.
<br>
<b> <u> Wiederherstellung von Verbesserungen </u> </b> - Möglichkeit, bei der Wiederherstellung aus einer Sicherung eine bestimmte Datei auszuwählen. Alle Sicherungsdateien auf Ihrem Gerät werden nun in einer Auswahlliste angezeigt, wenn Sie eine Wiederherstellung durchführen.
<br>
<b> <u> Android-Permission-Updates </u> </b> - Aktualisiert, um das neue Android 6.0-Berechtigungs-Framework zu unterstützen.
