<b><u> More Backup Enhancements </u></b> - Added support for partial backups. You can now backup a specific category by long-clicking the category and selecting "Backup this category".
<br>
<b><u> More Restore Enhancements </u></b> - Added ability to restore backups without overwriting existing data. You can now restore a backup into an existing category by long-clicking the category and selecting "Restore backup here".
